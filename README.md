# ViT (Vision Transformer)

Vision Transformer 관련 프로젝트입니다.

## 설명

이 프로젝트는 Vision Transformer 모델을 구현하고 실험하기 위한 저장소입니다.

## 설치

```bash
# 의존성 설치
pip install -r requirements.txt
```

## 사용법

```python
# 예제 코드
import torch
from vit import VisionTransformer

# 모델 생성
model = VisionTransformer()
```
